# Meat Processing Business

Base inicial do projeto da disciplina voltado para um dashboard mobile-first/PWA de acompanhamento de um frigorifico ficticio usando a API `https://bor.gs/tcc`.

## Documentacao inicial

- [Mapeamento de dados da API e proposta de negocio](./docs/api-dados-negocio.md)
- [Mapeamento frontend: telas, dados e derivacoes](./docs/frontend-mapeamento-dados.md)
- [Como rodar o frontend integrado](./docs/como-rodar-frontend.md)

## Observacao importante

Os dados reais da API sao de um laboratorio, nao de um frigorifico. A documentacao acima separa claramente:

1. o que pode ser reaproveitado diretamente da API;
2. o que pode ser reinterpretado para o contexto do frigorifico;
3. o que precisa ser criado localmente com dados sinteticos para suprir as telas de negocio.
